# Police Inter - Instructions de téléchargement

## Télécharger la police Inter

1. Rendez-vous sur : https://fonts.google.com/specimen/Inter
2. Cliquez sur "Download family"
3. Extrayez le fichier ZIP téléchargé
4. Copiez les fichiers suivants dans ce dossier (`assets/fonts/`) :
   - `Inter-Regular.ttf` (Weight 400)
   - `Inter-Medium.ttf` (Weight 500)
   - `Inter-SemiBold.ttf` (Weight 600)
   - `Inter-Bold.ttf` (Weight 700)

## Alternative : Téléchargement direct

Vous pouvez aussi télécharger directement depuis GitHub :
https://github.com/rsms/inter/releases

Prenez la version "Inter Desktop" et extrayez les fichiers TTF.

## Vérification

Une fois les fichiers copiés, vous devriez avoir :
```
assets/fonts/
├── Inter-Regular.ttf
├── Inter-Medium.ttf
├── Inter-SemiBold.ttf
└── Inter-Bold.ttf
```

Puis exécutez : `flutter pub get` pour appliquer les changements.
